# [Piedra Angular]

Este es un proyecto, el cual deseo utilizar como mi tarjeta de presentación, mi portafolio, para que puedan ver mis proyectos, mis habilidades, mi experiencia, etcétera. Incluso, dentro de él, viene mi currículum, para que lo visualice quien tenga interés en hacerlo, ya sea para trabajo, para darme un consejo o cualquier otro fin que resulte pertinente.

## 🚀 Tecnologías utilizadas
* **Lenguajes:** HTML5, CSS3, JavaScript (ES6+).
* **Herramientas:** Git, GitHub, VS Code (Antigravity), Netlify.

## 🛠️ Funcionalidades clave
* **Diseño Responsivo:** Adaptado para móviles, tablets y escritorio.
* **Formulario:** Para contacto directo con el usuario, enviando un correo a mi cuenta.
* **Integración con JavaScript:** Utilizado para crear un menú desplegable en dispositivos móviles y un modo oscuro para mayor accesibilidad.


## 📂 Cómo ejecutar el proyecto
1. Clona el repositorio o descárgalo y descomprímelo.
2. Abre el archivo `index.html` en tu navegador favorito.
